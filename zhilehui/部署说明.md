# 部署到云端的方法

## 方法一：Netlify Drop（最简单，推荐）

1. 访问：https://app.netlify.com/drop
2. 直接将 `index.html` 和 `questions.js` 两个文件拖拽到页面上
3. 等待几秒钟，会自动生成一个链接
4. 复制链接即可分享给他人

**优点**：无需注册，操作最简单

---

## 方法二：GitHub Pages（永久免费）

1. 注册 GitHub 账号：https://github.com
2. 创建一个新仓库（Repository）
3. 上传 `index.html` 和 `questions.js` 到仓库
4. 在仓库设置中启用 GitHub Pages
5. 获得链接：`https://你的用户名.github.io/仓库名/`

**优点**：永久免费，可以随时更新

---

## 方法三：Vercel（快速部署）

1. 访问：https://vercel.com
2. 使用 GitHub 账号登录
3. 点击 "New Project"
4. 上传文件或连接 GitHub 仓库
5. 自动部署并获得链接

**优点**：部署速度快，支持自动更新

---

## 方法四：CodePen（在线编辑）

1. 访问：https://codepen.io
2. 注册账号
3. 创建新的 Pen
4. 将 HTML 代码粘贴到 HTML 区域
5. 将 JS 代码粘贴到 JS 区域（需要将 questions.js 的内容内联）
6. 点击 "Save" 并分享链接

**注意**：需要将 questions.js 的内容合并到 HTML 中

---

## 推荐使用方法一（Netlify Drop）

这是最简单快速的方法，无需任何技术知识，拖拽即可完成！
